Drop your own HD graphic replacements (PNG, alpha honored) here.

The file names expected by each graphic slot are listed in ../manifest.ini
(the [replace:*] entries — e.g. `image = hd/title-logo.png`). Add a file whose
name matches a slot and it appears on the next launch; a slot with no matching
file keeps the original ROM graphics. See manifest.ini's header comment for how
to derive new replacement hooks with the in-game scene inspector (F3).

No graphics ship with this package. What you place here is yours to supply.
