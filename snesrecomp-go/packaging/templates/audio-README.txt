Drop your own OGG Vorbis music files here to replace the game's songs.

The file names expected by each song slot are listed in
../manifest.ini (the [music:*] entries — e.g. `file = audio/title.ogg`).
Add a file whose name matches a slot and it plays automatically on the next
launch; a slot with no matching file keeps the original ROM music. No editing
of manifest.ini is required to add music — only to retune loop points or gates.

No music ships with this package. What you place here is yours to supply.
